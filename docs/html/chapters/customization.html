Customization¶

The interface structure of QuOCS easily allows for user customization.

Add a new algorithm¶

To add a new algorithm in the QuOCS library, you need to install the software in the editable mode as explained in installation. Then, use the algorithm template to implement your own algorithm.

To easily use the name of your new custom algorithm in the optimization dictionary with the opti_algorithm_name key, you have to update the module_name and class_name keys in this file.

You can test your new algorithm, by using the examples in the tests folder or by creating your own test.

Add a new basis¶

The same procedure applies for the creation of new basis. A detailed guide for this procedure is also available here.
