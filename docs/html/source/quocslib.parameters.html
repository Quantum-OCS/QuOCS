class quocslib.parameters.TimeParameter.TimeParameter(map_index=- 1, time_name='time', initial_value=1.0, is_optimization=False, lower_limit=0.0, upper_limit=1.0, amplitude_variation=0.1)[source]

Bases: quocslib.parameters.BaseParameter.BaseParameter

Class for the time. In case the user chooses to optimize the time, the code redefines the time in function of an optimal parameter alpha: t = t_0 + alpha(t_max - t_min) where t_0 is the guess time, and t_max and t_min the limits for the optimal time. The limits for the optimal parameter alpha are then defined as: alpha_in = 0 alpha_max = (t_max - t_0)/(t_max - t_min) alpha_min = (t_min - t_0)/(t_max - t_min) alpha_variation = (amplitude_variation)/(t_max - t_min)
