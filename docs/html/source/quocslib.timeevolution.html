quocslib.timeevolution.piecewise_integrator.pw_evolution(U_store, drive, A, B, n_slices, dt)[source]¶

Compute the piecewise evolution of a system defined by the Hamiltonian H = A + drive * B and store the result in U_store

Parameters

  • U_store (List[np.matrix]) – the storage for all of the computed propagators

  • drive (np.array) – an array of dimension n_controls x n_slices that contains the amplitudes of the pulse

  • A (np.matrix) – the drift Hamiltonian

  • B (List[np.matrix]) – the control Hamiltonians

  • n_slices (int) – number of slices

  • dt (float) – the duration of each time slice

Return None
