functional(drive, A, B, n_slices, dt, U_store, rho_store, corho_store, sys_type)[source]¶

Compute the fidelity functional for the defined problem

Parameters

  • drive (np.array) – this should be a flat array that will be resized into N_ctrls x N_slices

  • A (np.matrix) – drift Hamiltonian

  • B (List[np.matrix]) – control Hamiltonians in a list of N_ctrls long

  • n_slices (int) – the number of pulse slices

  • dt (float) – the duration of each timeslice

  • U_store (List[np.matrix]) – a store for all the propagators

  • rho_store (List[np.matrix]) – a store for all the forward propagated states

  • corho_store (List[np.matrix]) – a store for all the reverse propagated states

  • sys_type (str) – either specifying state transfer or other

Return Tuple[float, np.array]

Returns a tuple containing the gradient and the figure of merit
