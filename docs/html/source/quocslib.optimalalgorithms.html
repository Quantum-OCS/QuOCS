The three important function are: * the constructor with the optimization dictionary and the communication object as parameters * run : The main loop for optimal control * _get_response_for_client : return info about the goodness of the controls and errors if any * _get_controls : return the set of controls as a dictionary with pulses, parameters, and times as keys * _get_final_results: return the final result of the optimization algorithm

functional(drive, A, B, n_slices, dt, u0, rho0, rhoT, sys_type)[source]¶

Compute the fidelity functional for the given problem defined in the class

Parameters

  • drive (jnp.array) – a flat array that contains the pulse amplitudes

  • A (jnp.matrix) – the drift hamiltonian

  • B (List[jnp.matrix]) – the control hamiltonians

  • n_slices (int) – the number of slices in the pulse

  • dt (float) – the duration of each timeslice

  • u0 (jnp.matrix) – the initial propagator that should be used

  • rho0 (jnp.matrix) – the initial density matrix

  • rhoT (jnp.matrix) – the target density matrix

  • sys_type (str) – the string to indicate the system type, can be either StateTransfer or left blank

Return float

the value of the fidelity at the current point in time

functional(drive, A, B, n_slices, dt, U_store, rho_store, corho_store, sys_type)[source]¶

Compute the fidelity functional for the defined problem

Parameters

  • drive (np.array) – this should be a flat array that will be resized into N_ctrls x N_slices

  • A (np.matrix) – drift Hamiltonian

  • B (List[np.matrix]) – control Hamiltonians in a list of N_ctrls long

  • n_slices (int) – the number of pulse slices

  • dt (float) – the duration of each timeslice

  • U_store (List[np.matrix]) – a store for all of the propagators

  • rho_store (List[np.matrix]) – a store for all of the forward propagated states

  • corho_store (List[np.matrix]) – a store for all of the reverse propagated states

  • sys_type (str) – either specifying statetransfer or other

Return Tuple[float, np.array]

Returns a tuple containing the gradient and the figure of merit
