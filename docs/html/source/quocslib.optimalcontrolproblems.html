Abstract method for figure of merit evaluation. It returns a dictionary with

Parameters

  • pulses_list (list) – List of np.array. Every np.array is a pulse.

  • time_grids_list (list) – List of np.array. Every np.array is a time grid at each time grid corresponds

to a pulse. :param list parameters_list: List of floats. :return dict: The dictionary must contain at least the “FoM” key with the figure of merit float. Other possible keys can provide useful information for errors or other.
