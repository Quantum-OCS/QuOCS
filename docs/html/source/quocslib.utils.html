An evaluation method for the figure of merit via files exchange. The communication object accesses to the get_FoM function. The get_FoM removes the "fom.txt" file and creates a json, a txt or a npz file in the controls folder designed by the user. Finally, the read_fom_values will wait and check for the figure of merit in the folder designed by the user.
